Getting Started

Open a Command Prompt window and change the directory to the location that you want to save your Meteor project to. Keep in mind that when you create a Meteor project, it will make a new folder and automatically create the files that you will need to make your website. Now, type:

meteor create myapp

Here, "myapp" represents the name of your project. You should replace all references to "myapp" with the name of your project. If you change to the new myapp directory, you will see a .meteor folder and the following three files: myapp.css, myapp.html, and myapp.js. These are the files that you will be modifying to create your website. Meteor includes some sample code in these files so that you can see a working website immediately. To start the Meteor server and look at the sample website, make sure your Command Prompt window is looking at your project folder and type meteor to start up your Meteor server. Then, you can go to your web browser and type http://localhost:3000 into the address bar. This will show you the sample website that Meteor gives you when it makes your project files. If you keep this window open, you can check back frequently to see the changes and progress you are making on your website. It will also show any errors that occur as a result of problems with your code.
