Install Meteor

To start out using Meteor, you must first go to this website to learn how to install Meteor for your operating system. I am using Windows, so I downloaded the installer and ran it. As part of installing Meteor, you have the option to create a developer account that allows you to deploy your app with their free service. It will also allow Meteor to notify you about any critical security updates. As part of the installation process, you also have to allow firewall access. Once Meteor is installed, you are ready to get started.
